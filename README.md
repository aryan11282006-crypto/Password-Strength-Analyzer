# Password Strength Analyzer 🔐

This is a simple web-based Password Strength Analyzer built using HTML, CSS, and JavaScript.

## 🚀 Features
- Show/Hide Password
- Weak / Medium / Strong detection
- Live password validation
- Strong password suggestions
- Security tips

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
